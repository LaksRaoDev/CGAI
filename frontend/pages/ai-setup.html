Install Dependencies

Install the required Python packages for AI functionality.

Run in your backend directory:

Or install all requirements:

pip install -r requirements.txt
4

Restart Backend Server

Restart your backend server to apply the changes.

Stop current server (Ctrl+C) and restart:

python app.py

Troubleshooting

Backend server not starting?

Make sure you have Python 3.8+ installed and all dependencies are installed correctly.

API key errors?

Double-check your API key is correct and you've saved the .env file properly.

Still using templates?

The system will automatically fallback to templates if AI is not available. This ensures your app always works.
